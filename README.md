## Installation

Install and run project with npm

```bash
  git clone https://github.com/MezianeKhalil/redux-shopping-cart.git
  cd redux-shopping-cart
  npm install
  npm run dev
```
    
## Tech Stack

**Client:** React, Redux, TailwindCSS

**Server:** Fake Store API
